# Premium Portfolio

A polished React portfolio template built with Vite, Tailwind CSS, and Framer Motion. It includes a multi-section home page, resume page, and reusable UI components designed for showcasing skills, projects, experience, certifications, testimonials, and contact information.

## Features

- Responsive portfolio layout with animated page transitions
- Hero, about, skills, projects, experience, certifications, resume, testimonials, and contact sections
- Smooth scroll progress indicator and back-to-top button
- Custom theme support with a theme toggle component
- Resume page ready for download with a placeholder printable layout
- Lightweight React Router navigation with lazy-loaded routes

## Technologies

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- React Router DOM
- React Icons
- EmailJS browser integration

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Folder Structure

- `src/` — main application source files
- `src/components/` — reusable UI components
- `src/sections/` — page sections for the portfolio layout
- `src/pages/` — route-level pages like home and resume
- `src/layouts/` — application layout and navigation
- `src/data/` — JSON files for content data
- `src/hooks/` — custom React hooks

## Notes

Replace placeholder data in `src/data/` and section components with your own profile, work experience, projects, and resume details to customize the portfolio.
